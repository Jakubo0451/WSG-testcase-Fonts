Broken WSG 2.18 Font Optimization

Environment

Minimizing both the quantity and complexity of custom fonts implemented on a website significantly decreases the amount of data that needs to be transferred to the user's device and reduces the computational effort required for rendering, which in turn contributes to lower energy consumption and helps reduce the associated carbon emissions during page loads.

Performance

Choosing to use well-optimized web fonts or default system fonts plays a key role in enhancing overall website performance by enabling faster text rendering and contributing to a more seamless and responsive user experience, especially on devices with limited processing power or slower internet connections.

Social equity

System fonts, which come preinstalled on most operating systems and devices, are a highly reliable choice for web design because they eliminate the need for additional downloads and ensure that content is displayed quickly and consistently in a typeface that users are already accustomed to, improving both performance and readability.
